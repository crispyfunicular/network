Cookies técnicas (siempre activas)

Ver listado detallado de este tipo de cookies

   Denominación Origen Duración Finalidad
   UMB_UCONTEXT Umbraco Persistente, finaliza al terminar la sesión. Esta
   es una cookie específica de Umbraco. Almacena una referencia de Guid al
   usuario backend registrado actualmente. Generado al azar al iniciar
   sesión y almacenado en la base de datos, le permite acceder al usuario
   actual, sin tener que almacenar ningún dato específico del usuario en
   la cookie.
   UMB_UPDCHK Umbraco Persistente, finaliza al terminar la sesión. Esta es
   una cookie específica de Umbraco. Almacena un flag para indicar que se
   ha realizado una comprobación de actualización. Cuando una cookie
   caduca o no se encuentra una cookie, se realiza otra comprobación de
   actualización (configurando la cookie nuevamente). Las comprobaciones
   solo se realizan si el usuario actual es un administrador de fondo.
   Crtmwidgetcookies CRTM Persistente Identifica si se ha realizado la
   aceptación de las cookies en el widget.
   crtmwidgetProtDatos CRTM Persistente Identifica si se ha realizado la
   aceptación de la protección de datos en el widget.
   crtmcookiesSVProtDatos CRTM  Persistente Identifica si se ha realizado
   la lectura y aceptación de la política de protección de datos de
   solicitud de visita en el Portal Web.
   crtmcookiesProtDatos CRTM  Persistente Identifica si se ha realizado la
   aceptación de las cookies y la política de protección de datos en el
   Portal Web.
   crtmcookiesCAnaliticas CRTM Persistente Indica si se han aceptado las
   cookies de tipo Analítico.
   crtmcookiesCPublicitarias CRTM Persistente Indica si se han aceptado
   las cookies de tipo Publicitario.
   crtmcookiesCPersonalizacion CRTM Persistente Indica si se han aceptado
   las cookies de tipo Personalización.

   permiten elaborar un perfil de usuario.

     * Inclusite (Herramienta de accesibilidad)
     * Los mapas de líneas que usen Google.
     * Google Analytics

Ver listado detallado de este tipo de cookies

   Denominación Origen Duración Finalidad
   __utma Google Persistente, dos años a partir de la configuración o
   actualización. Se usa para distinguir usuarios y sesiones. La cookie se
   crea cuando se ejecuta la biblioteca JavaScript y no hay ninguna cookie
   __utma. La cookie se actualiza cada vez que se envían datos a Google.
   __utmb Google Persistente, 30 minutos a partir de la configuración o
   actualización. Se usa para determinar nuevas sesiones o visitas. La
   cookie se crea cuando se ejecuta la biblioteca JavaScript y no hay
   ninguna cookie __utmb. La cookie se actualiza cada vez que se envían
   datos a Google.
   __utmc Google Hasta fin de la sesión del navegador. Se configura para
   interactuar con urchin.js. Anteriormente, esta cookie actuaba junto con
   la cookie __utmb para determinar si el usuario estaba en una nueva
   sesión o visita.
   __utmt Google Persistente, diez minutos a partir de la configuración o
   actualización. Se usa para limitar el porcentaje de solicitudes.
   __utmv Google Persistente, dos años a partir de la configuración o
   actualización. Se usa para almacenar datos de variables personalizadas
   de visitante. Esta cookie se crea cuando un programador usa el método
   _setCustomVar con una variable personalizada de visitante. También se
   usaba para el método _setVar retirado. La cookie se actualiza cada vez
   que se envían datos a Google.
   __utmz Google Persistente, seis meses a partir de la configuración o
   actualización Almacena la fuente de tráfico o la campaña que explica
   cómo ha llegado el usuario al sitio. La cookie se crea cuando se
   ejecuta la biblioteca JavaScript y se actualiza cada vez que se envían
   datos a Google.
   _ga Google Persistente, dos años a partir de la configuración o
   actualización. Se usa para distinguir a los usuarios.
   _gat_gtag_[UID] Google Persistente, finaliza al terminar la sesión.
   Incluye información de la campaña relativa al usuario. Si has vinculado
   tus cuentas de Google Analytics y AdWords, las etiquetas de conversión
   de sitios web leerán esta cookie, a menos que la inhabilites.
   _gid Google Persistente, diez minutos a partir de la configuración o
   actualización. Se usa para distinguir a los usuarios.
   INCLUSSID Google Sesión Cookie de sesión de Inclusite, herramienta de
   apoyo en navegación web para personas con discapacidad. Caduca al
   finalizar la navegación iniciada por el usuario para facilitar la
   comunicación usuario/servidor.

La Red de Cercanías ferroviarias

   La red ferroviaria de Madrid, nacida a mediados del siglo XIX, se ha
   convertido en una moderna red de Cercanías de 391 km de longitud, 92
   estaciones y 9 líneas, que transporta 820.000 viajeros diarios.
